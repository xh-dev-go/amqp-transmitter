# amqp-transmitter

## install
```
go install github.com/xh-dev-go/amqp-transmitter
```

## build
```
go build
```

## Command
```shell
$ ./amqp-transmitter
  -amqp-url string
        The connection string of amqp
  -queue-name string
        The name of the queue
  -version
        show application version

# Demo
./amqp-transmitter --amqp-url {url} --queue-name {queue name}
```
